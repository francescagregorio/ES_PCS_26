## 👥 Progetto finale (C++)
**Titolo**: Implementazione del Metodo delle Correnti di Maglia per la risoluzione di circuiti elettrici\
**Componenti del gruppo**: Noemi Fantini, Francesca Gregorio, Mariarosa Labella\
**Descrizione**: il progetto consiste nello sviluppo di un programma per la risoluzione automatica di circuiti elettrici mediante il metodo delle correnti di maglia. A partire dalla topologia del circuito (nodi, rami, componenti e relativi valori), il software individua le maglie indipendenti, costruisce il sistema di equazioni secondo la legge di Kirchhoff delle tensioni e lo risolve per determinare le correnti circolanti in ciascuna maglia. Da queste è poi possibile ricavare le correnti e le tensioni su ogni singolo ramo del circuito.
